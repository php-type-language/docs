# @phpstan-implements

<primary-label ref="phpdoc-component"/>
<secondary-label ref="alias-tag"/>

The `@phpstan-implements` tag is PHPStan's vendor-specific spelling of
[@implements](implements-tag.md), which this component already recognizes
bare.

This library recognizes it as an alias: it parses identically to the
canonical tag above and produces the exact same tag instance, keeping the
vendor-prefixed name it was written with.

Documented by
[PHPStan](https://phpstan.org/writing-php-code/phpdocs-basics).
